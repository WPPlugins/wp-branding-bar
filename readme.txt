== Description ==

WP Branding Bar creates an adjustable header bar accross the top of your site, with room for two user-defined images with two user-defined links.  Provides default images and branding for sites in the utexas.edu domain.

By default the branding bar has the following characteriestics:

* Appears on every public facing page of your site
* Appears below the WordPress admin bar
* White background
* 40px tall
* Covers the width of the viewport
* Includes a (very ugly) left-aligned and a (very ugly) right-aligned image
* Each image is aligned to the width of the main content div in your site
* Each image is a link to example.com

Defaults that can be changed on the settings page

* Height of header bar
* Width of the div that contains the branding images
* The branding images
* The url the branding image points to
* The background image of the div

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload and unzip `wp-branding-bar.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add your own branding images and links on the Settings --> WP Branding Bar settings page
